geocodeStreetAddress:city:state:zipCode:country:inRegion:completion:

Geocodes address. City/State/Zip/Country/Region are all optional. If you don’t provie a country, will default to ‘United States’. Returns an array of XGeocoder results.

reverseGeocodeCoordinate:completion:

According to Apple, you can only have one reverse geocoding request at any given moment, so calling this will cancel any exisiting requests (completion block will return nil).
